Flick through our projects

Guardian News

Produced a news summary site drawing in the latest headlines from the Guardian API. Reused functions and pooled DOM insertions to minimize page loading time. Created tabbed interface and ability to dynamically add new content sections on the fly

Twitter Feed

Built a network of Node servers to query Twitter's API and render selected images into a gallery webpage, returning only tweets matching selected tags. Archived data in our own database. Built second server to render images from our database into a gallery webpage, utilizing caching.

Dynamic Data

Created a dynamic data visualisation map with D3 and topoJSON to illustrate cocaine salt prices worldwide. Buttons allow the user to toggle between wholesale and retail prices instantly, and a tooltip shows specific data on any country of interest.

Blog Platform

A RESTful blogging platform using Hapi: multiple authentication options, including Facebook and Twitter. Jade templating allows dynamic pages to be shown depending on permissions, while a MongoDB database handles entries and comments on entries instantly. Hosted on the cloud
